Vulnerability Scanning

Introduction

Vulnerability scanning is the process of discovering and identifying vulnerabilities in a network by using Vulnerability Scanners (OpenVAS, Nessus, Nmap, Metasploit) and scripts.

Vulnerability Scanning with Nmap

Nmap(“Network Mapper”) is a free and open source utility for network exploration and security auditing. It is most popular network security scanner that has advanced features like remote OS detection, version/service detection, IP ID idle scanning, the Nmap Scripting Engine(NSE) and fast multi-probe ping scanning. The Nmap Scripting Engine (NSE) uses a collection of special-purpose scripts to gain even more information about remote systems. NSE is powered by the Lua programming language and a standard library designed for network information gathering. NSE offered advanced feature like version detection, notification of service vulnerabilities, and discovery of backdoors and other malware. NSE scripts are located in the /usr/share/nmap/scripts directory.

We navigate to Nmap NSE Scripts /usr/share/nmap/scripts in zsh while cd /usr/share/nmap/scripts in bash. we get vulnerability scripts by run this command ls *vuln*
