THE commands that you need to know

Linux Commands

ifconfig

  • Display or configure network interface parameters.

    ifconfig
ifconfig <interface> up/down

ip

  • A more versatile tool for network configuration.

    ip addr show
ip link set <interface> up/down

route

  • Display or manipulate the IP routing table.

    route -n
route add default gw <gateway>

ping

  • Test network connectivity.

    ping <hostname/IP>

traceroute

  • Trace the route that packets take to reach a destination.

    traceroute <hostname/IP>

netstat

  • Display network connections, routing tables, interface statistics, masquerade connections, and multicast memberships.

    netstat -an

nmap

  • Network exploration tool and security/port scanner.

    nmap <target>

tcpdump

  • Capture and analyze packets on a network.

    tcpdump -i <interface>

arp-scan

  • Scans for live hosts on a network using ARP.

    arp-scan <network_range>

iwconfig

  • Configure wireless network interfaces.

    iwconfig

ss

  • Display socket statistics.

    ss -tunapl

host

  • Perform DNS lookups.

    host <hostname>

dig

  • A versatile DNS query tool.

    dig <hostname>

iptables

  • Configure IP packet filter rules.

    iptables -L

Wireshark

  • A powerful network protocol analyzer (requires installation).

    wireshark

hping3

  • A versatile tool for packet crafting and sending.

    hping3 -c 3 -S -p 80 <target>

ettercap

  • A comprehensive, mature suite for man-in-the-middle attacks.

    ettercap -T -q -M arp:remote /<gateway>// /<target>//

aircrack-ng

  • A suite of tools for assessing Wi-Fi network security.

    airodump-ng <interface>

tcpdump with filters

  • Capture packets with specific criteria.

    tcpdump -i <interface> host <target_ip> and port <target_port>

nload

  • Display incoming and outgoing traffic separately.

    nload <interface>

Snort

  • A powerful open-source intrusion prevention system (IPS) and network intrusion detection system (NIDS).

    snort -q -A console -q -c /etc/snort/snort.conf -i <interface>

Burp Suite

  • A graphical tool for testing web application security.

    java -jar burpsuite_community_<version>.jar

sqlmap

  • An automated SQL injection and database takeover tool.

    sqlmap -u "http://example.com/index.php?id=1" --dbs

mitmproxy

  • An interactive TLS-capable intercepting proxy.

    mitmproxy

John the Ripper

  • A fast password cracker.

    john --wordlist=passwords.txt --format=nt <hashfile>

Hydra

  • A fast and flexible password-cracking tool.

    hydra -l <username> -P <passwords_file> <target> <protocol>

Aircrack-ng

  • A suite of tools for auditing wireless networks.

    aircrack-ng -w <wordlist> -b <BSSID> <capture_file.cap>

Recon-ng

  • A full-featured web reconnaissance framework.

    recon-ng

THC-Hydra

  • Another versatile online password cracking tool.

    hydra -l <username> -P <passwords_file> -e nsr -t 4 -u <target> <protocol>

Metasploit Framework

  • A powerful penetration testing framework.

    msfconsole

Sniffing with Scapy

  • A powerful interactive packet manipulation program.

    scapy

Nikto

  • A web server scanner that performs comprehensive tests against web servers for multiple items.

    nikto -h <target>

Dirb

  • A web content scanner, often used for finding hidden directories.

    dirb http://<target>

Wafw00f

  • Identifies and fingerprints Web Application Firewall (WAF) products.

    wafw00f <target>

tcpick

  • A text-based TCP packet sniffer.

    tcpick -C -yP -r <pcap_file>

Windows Commands

ipconfig

  • Display or configure network interface parameters.

    ipconfig

netsh

  • Network shell command-line tool.

    netsh interface ipv4 show addresses
netsh interface ipv6 show addresses

netstat

  • Display network connections, routing tables, interface statistics, masquerade connections, and multicast memberships.

    netstat -an

route

  • Display or manipulate the IP routing table.

    route print
route add 0.0.0.0 mask 0.0.0.0 <gateway>

ping

  • Test network connectivity.

    ping <hostname/IP>

tracert

  • Trace the route that packets take to reach a destination.

    tracert <hostname/IP>

nslookup

  • Query DNS for information.

    nslookup <hostname>

arp

  • Display and modify the ARP (Address Resolution Protocol) cache.

    arp -a

nmap

  • Network exploration tool and security/port scanner (can be installed on Windows).

    nmap <target>

ncat

  • A feature-rich networking utility that reads and writes data across networks using the TCP/IP protocol.

    ncat -lvp 4444

curl

  • A powerful tool that crawls, download and analyse website

    curl <hostname/ip>

Git Commands

  • The command displays git versions

    git --version

  • Initailizing a repository

    git init

  • Staging files

    git add file.java #single file
git add file.java file.py #multi files
git add *.java #using pattern or extension
git add .

  • Committing staged files

    git commit -m "[message]"
git commit #open default editor

  • Removing the staged files

    git rm file.java
						git rm -r folder #folder with subfolders and files

  • Moving and Renaming a file

    git mv file.go userinput.go

  • Viewing the status

    git status # full
git status -s # short

  • configuring - get and set repository or global options

    git config --global user.name <full name>
						
    git config --global user.email <email>
						
    git config --global core.editor "code/subl wait"
						
    git config --global -e
						
    git config --global core.eol=crlf
git config --global core.eol=lf
						
    git config --global core.autocrlf true #for windows users
						
    git config --global core.autocrlf input #for mac & linux users

Assembly

  • Structure and write a "Hello, World!" program in Assembly

    section .data
	msg db "Hello World!", 0x4
	len equ $-msg

global _start

section .text
_start:
	mov eax, 4      ; 
	mov ebx, 1      ;
	mov ecx, msg    ;
	mov edx, len    ; 
	int 0x80        ;

	; exit 
	mov eax,1
	int 0x80
